## Descrição.
 
 Projeto para estudos, para revisar estrutura do nest js.

Libs externas:

            swagger;
            class-validator;
            classe-transforme;

## Instalação das dependencias.

```bash
$ npm install
```

## Compilar e rodar o projeto.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rodar testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
